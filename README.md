# pico_v2_arduino_examples

![pico_v2](https://rt-net.github.io/images/picov2/pico_v2.jpg)

Pi:Co V2用のArduinoサンプルスケッチ集です。

## 動作環境
- Arduino IDE v2.3.4
- arduino-esp32 v3.1.1
- Async TCP 3.3.6
- ESP Async WebServer 3.7.2

## サンプルスケッチについて

### STEP8
- 超信地旋回でマイクロマウスとして動作するサンプル
### STEP9
- 超信地旋回で速度と加速度を選択してマイクロマウスとして動作するサンプル
### STEP10
- STEP9に既知区間加速を追加してマイクロマウスとして動作するサンプル
### STEP11
- スラロームの基礎を追加してお試しで旋回するサンプル
### STEP12
- STEP10にスラロームの探索を追加してマイクロマウスとして動作するサンプル
### STEP13
- STEP12にスラロームの最短を追加してマイクロマウスとして動作するサンプル
### STEP14
- 最短走行をパターン化のサンプル
### STEP15
- step13に最短走行のパターン化を追加してマイクロマウスとして動作するサンプル
### STEP16
- 大回りターンの基礎追加してお試しで旋回するサンプル
### STEP17
- step15に大回りターンを追加してマイクロマウスとして動作するサンプル
